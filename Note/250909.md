     - 현재 위치에서 뚫려있는 터널로만 이동 가능

3. 시작점부터 주변으로 점점 퍼져나가면서 확인(BFS)
   - Queue를 활용해서 먼저 확인하는 노드부터 먼저 확인
   - 먼저 방문한 노드에서 갈 수 있는 노드들을 후보군(Queue)에 추가
4. BFS의 시간 복잡도
   - O(V + E)
     - V: 정점의 개수, E: 간선의 개수
   - 정점의 개수 = 2,500개(50 * 50)
     - Queue에 2,500개 까지 들어갈 수 있음
     - -> 여유로움
   - 간선의 개수 = 2500 * 4(상하좌우) = 10,000개
